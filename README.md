# ⏳ Tiiime

> Your personal, crowd-sourced timetable

Built for **Byld Hackathon v16**.

## Instructions

The app is live on [**tiiime.now.sh**](https://tiiime.now.sh).

### Development

1. Clone repo
2. `$ npm install`
3. `$ node index`

The app should now be available on port 3000 of your localhost.
